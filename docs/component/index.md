# MQ-Responsive Component

For convenience, an `<MqResponsive>` component is available for use in your app. You can import it like so:

::: code-group

```vue [Composition API]
<script setup>
import { MqResponsive } from "vue3-mq";
</script>
```

```vue [Options API]
<script>
import { MqResponsive } from "vue3-mq";

export default {
	components: {
		MqResponsive,
	},
};
</script>
```

:::
